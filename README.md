# 🍪 Cookie Clicker (Python Tkinter)

A simple Cookie Clicker game made with **Python and Tkinter**.

Click the cookie to earn cookies, buy upgrades to increase your cookies per click, and watch your cookie empire grow!

## 🎮 Features

- 🍪 Click the cookie to earn cookies
- ⬆️ Multiple cookie upgrades
- 💾 Automatic save system
- 🖼️ Custom cookie images for each upgrade level
- 🖥️ Simple graphical interface using Tkinter

## 🚀 Installation

### Requirements

- Python 3.x
- Tkinter (usually included with Python)

### Download

Clone the repository:

```bash
git clone https://github.com/yourusername/Cookie-Clicker.git
```

Go into the project folder:

```bash
cd Cookie-Clicker
```

Run the game:

```bash
python main.py
```

## 🛒 Upgrades

| Upgrade | Effect |
|---|---|
| 🍪 Normal Cookie | +1 cookie per click |
| 🟢 Pea Cookies | +5 cookies per click |
| 🌱 Plant Cookies | +25 cookies per click |
| 🔥 Firey Cookies | +125 cookies per click |
| ❄️ Icey Cookies | +525 cookies per click |

## 💾 Save System

The game saves:

- Current cookies
- Cookies gained per click
- Upgrade prices

Your progress will still be available after closing and reopening the game.
